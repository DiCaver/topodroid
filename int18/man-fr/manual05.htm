[5] DONNÉES d'étalonnage du DistoX

Valable uniquement pour DistoX, DistoX2 et DistoXBLE

La Fenêtre d'étalonnage affiche la liste des données de d'étalonnage. Si la base d'étalonnage de contient aucune données, la liste est vide, mais les bouton du haut sont disponibles. Chaque ligne de donnée contient les informations suivantes :

  • index: index incrémental qui identifie la donnée
  • groupe: numéro du groupe à qui appartient la donnée
  • erreur: la différence entre la direction corrigée par l'étalonnage pour la direction, la direction moyenne du groupe (en degrés)
  • azimuth, pente et roulis, calculé depuis les valeur brute sans correction d'étalonnage
  • optionnellement, les données brute d'étalonnage ; ex: la composante G et M dans la trale DistoX en valeurs décimales ou hexadécimales.

Couleurs

Les données d'étalonnage sont affichée en jaune et un texte bleu, avec une alternance paire et impaire des numéros de groupes. Les données sans groupe sont grises.

Les couleurs de fond pour les donénes sont :

  • noir : valeurs normales
  • violet : données incomplètes (non utilisées pour l'étalonnage, mais utilisées pour le groupe)
  • maron : données saturées
  • gris : données supprimées (seulement si affichées)
  • rouge : données dint l'erreur est supérieure à 1°
  • vert : données différant de la valeur moyenne du groupe pour une valeur supérieure à la valeur distance de groupe

ACTIONS

  • Appuie sur une donnée : ouvre la fenêtre d'édition

BOUTTONS

  • bascule le DistoX en mode étalonnage (cela peut échouer au premier essai
  • Réinitialise la connexion BT ou contrôle de DistoX à distance
  • Télécharge les données d'étalonnage du DistoX
  • affecte un numéro de groupe aux données, ou réinitialise les numéros de groupe
  • Calcule les coefficients d'étalonnage
  • distribution de la direction des données d'étalonnage
  • lit et affiche les coéficients d'étalonnage stockés dans le DistoX
  • écrit les coéficients d'étalonnage dans le DistoX. Activé uniquement après le calcul des coéficients. Les coéficients peuvent également être écrit depuis la fenêtre de résultats de calculs. Un avertissement in affiché sir la couverture des directions est inférieure à 95% ou si l'erreur moyenne est supérieure à 0.5°
  • ouvre la fenêtre de recherche

Pour le moment, seul le téléchargement de données A-la-demande n'est activé pour les données d'étalonnage. Les données téléchargées sont ajoutées à l'étalonnage actuel. Par conséquent, vous pouvez prendre quelques mesures, télécharger les données, vérifier si elles sont satisfaisantes, si non, en prendre d'autres et les télécharger de nouveau. Et cela, jusqu'à trouver satisfaction dans vos données d'étalonnage. Les données télécharger peuvent être "désactivées" en définisant le numéro de groupe à "0" (zéro). Les données désactivées ne sont pas utilisées dans le calcul des coéficients d'étalonnage. Elles ne sont pas non plus supprimées (normalement, elle ne sont pas affichée dans la liste). Les données supprimées ne sont pas prise en compte dans affectations des numéros de groupe et dans le calcul des coéficients.

Bluetooth

Ce bouton réinitialise la connexion Bluetooth de TopoDroid.
Il permet également à l'utilisateur de contrôler le DistoX à distance [A] : éteindre ou alumer le laser et prendre des visées pour l'étalonnage. Les données sont imédiatement téléchargée si le mode est défini sur "Continu".
Dans ce cas, il est également possible de prendre (et télécharger) un groupe de quatre visées d'étalonnage avec un court délai entre elles (controlé par le paramètre "attente laser" et attente visée").

Le bouton Bluetooth est désactivé tant que TopoDroid prend des visées d'étalonnage.

Calculs de l'étalonnage

Le bouton "roue dentée" démarre le calcul des coéficients d'étalonnage.
Quand le calcul se termine, la fenêtre de coéficients est affichée.
Si il n'y a pas sufisament de données ou de groupe de données, le calcul n'éboutit pas et un avertissement est affiché.
L'étalonnage échoue si la limite d'itération est atteinte avant la fin du calcul. Cela arrivé généralement si les numéros de groupes sint affectés de manière erronné. Dans cette situation, un message est également affiché.

Avertissements

  • données saturées: données avec un valeur de champ magnétique anormal. Une cause possible est la présence d'un champ magnétique anormal. Les données saturées possède une couleur de fond marron
  • Aucune données : les données d'étalonnage reçues incorectement. La donnée ne possède pas de valeurs et par défaut n'est pas assignée à un nombre. Toutefois, elle est utilisée pour le calcul des groupes. Les données nulles son affichées avec un fond violet

MENUS

  • Afficher : affiche ou non toutes les données, incluant celles supprimées
  • Valider : validation des données d'étalonnage avec une autre
  • Paramètres : identique à la fenêtre d'étalonnage
