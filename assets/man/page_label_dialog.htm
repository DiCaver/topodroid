DRAWING LABELS

This dialog is opened when you insert a label, in drawing mode of the sketch window.

Label properties:

  • size: one of XS, S, M, L, XL
  • text
  • layers, canvas views [T]
  • additional Therion options

The canvas views are presented as checkboxes with the view letter-code. By default labels belongs to the "base" and "artifact" views. It is possible to switch off some views. Beware that, if the label is not assigned to a visible view, it will not appear in the canvas even if it is actually added to the sketch.

Labels are orientable.
