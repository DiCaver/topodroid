GEO-POINTS LIST

This window is opened tapping the GNSS button in the survey info window.

The window shows the list of geo-localized points, and has three buttons to insert a new point.

  • GNSS gets the new point coordinates with the GNSS (if "Location" permission has been granted)
  • Add enter the coordinates of the new geo-point manually
  • Import the coordinates from a point file created by a GNSS app
  • start GpsTest app

Menus:

  • Close the window
  • Change the external geopoint app
  • Help

External GNSS apps
TopoDroid can import geopoint coordinates from

  • MobileTopographer point files. A dialog is opened to choose the geopoint.
  • GPX recorder track (gpx) file. The last track-point is used.
  • GPX position position (csv) file. TopoDroid computes an average of the positions.

The station name of geopoint imported from GPX recorder and GPX position is set by TopoDroid automatically as a number prefixed by '#'.

Furtermore geopoint coordinates saved in the clipboard with the GpsTest app can be transferred to the fields in the insert dialog. Make sure to set the checkbox to store also the heigth in the clipboard.

Tapping a geo-point entry in the list opens the Geo-point edit dialog.

Internally TopoDroid stores the coordinates in decimal degrees. In the coordinate dialogs you can type them either in decimal degrees or as "dd:mm:ss.ss". The TopoDroid keypad has special keys for "degree" and "minute". If you use a keyboard that does not have semicolon ':', use the space instead. If the keyboard does not have the decimal point '.' use the slash or the comma instead.

Location settings

  • Location units: either dd.mm.ss or decimal degrees [A]
  • C.R.S. coordinate reference system [default is long-lat] (disabled)
  • Geopoint app [default none]
