DistoX NICKNAME

This dialog is opened long-tapping on a device entry in the device window.

It displays some info about the selected DistoX and has a text field to enter the nickname

. If the second-distox setting is enabled there is a checkbos to set the DistoX as second device.

OK save the nickname to the database and closes the dialog.
If the checkbox is selected the DistoX is set as second device.
